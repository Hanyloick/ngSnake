# ngSnake

## Built With

[Angular CLI](https://angular.io/docs)

## Installation

1. Clone the Repo
    `https://github.com/Hanyloick/ngSnake.git`

2. Install NPM packages
    `npm install`
3. Run
    `ng serve -o`

## Controls

>T to start game and R to restart.
WASD or Arrow Keys to change direction.
